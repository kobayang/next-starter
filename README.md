# Next-starter

Next.js starter kit

## Using libraries

- Next.js
- TypeScript
- React
- Redux 
  - Redux-thunk
  - typescript-fsa and related typescript-fsa-* 
- Styled-components

*NOTE: bump library versions to latest on 2018/08/08*

## Starting

```bash
git clone git@github.com:kobayang/next-starter.git
yarn install
yarn run next build
yarn run next start
rm -rf .git # If you want to remove this git histries.
```
